📘 BonusSchool – School Management System
📌 Overview

BonusSchool is a desktop-based School Management System developed using C# and SQL Server.

This project demonstrates basic CRUD operations, relational database management, and data integration within a school automation scenario.

🚀 Features

Student Management

Teacher Management

Lesson Management

Club Management

Grade Tracking

SQL JOIN queries

Data filtering by student ID

🛠️ Technologies

C# (.NET Framework)

Windows Forms

SQL Server

ADO.NET

🎯 Purpose

The main goal of this project was to strengthen my understanding of:

Database relationships

Parameterized SQL queries

Data binding with DataGridView

CRUD operations
